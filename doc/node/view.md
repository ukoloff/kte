# Утилита просмотра списка КТЭ

Данная Web-утилита предназначена для
просмотра [списка КТЭ][xml],
сгенерированного
[алгоритмом распознавания](../rec/),
в отладочных целях.

[xml]: ../rec/RecognitionResultFileFormat.docx

## Сборка

Сборка, тестирование и установка утилиты
производятся одновременно с [другими модулями](.),
использующими язык JavaScript
на платформе Node.js

## Использование модуля

- Откройте сгенерированный файл `index.html`
  в любом современном браузере
  + Доступен также по адресу https://ukoloff.github.io/kte/
- Выберите пункт "`Просмотр результатов распознавания КТЭ`"
- Нажмите кнопку "`Загрузить результат распознавания!`" и
  укажите файл в формате XML
  + Можно просто натащить файл на Web-страницу,
  он также откроется
- Если исходный файл корректен,
  откроется окно,
  содержащее изображение контура детали
- КТЭ будут выделены разными цветами
- Окно можно масштабировать при помощи колеса мыши
- Окно можно двигать при помощи мыши
- При наведении указателя на КТЭ,
  оно будет выделено путём анимации,
  показывающей направление его обработки,
  и во всплывающем окне показаны его свойства:
  + Нумерация
    `id` и `parent_id`
  + Вид КТЭ
    `side`, `pos`, `type` и `subtype`
  + Некоторые геометрические характеристики
    `A`, `B`, `Rmin`, `X`, `Z`...
