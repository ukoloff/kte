Для построения приложения:

1. Открыть файл source\NRecognition.sln в Visual Studio 2019. При запросе установить необходимые дополнительные компоненты.
2. Установить NPRecognition стартовым проектом
3. Установить конфигурацию Release
3. Построить приложение. В папке Release все файлы, необходимые для запуска: BRecognize.dll, NCMb.dll, NPRecognition.exe

Запуск и использование:

Формат командной строки:

NPRecognition <имя входного файла> <имя выходного файла>

Код возврата: 0 - успешно; 1 - произошла ошибка.
